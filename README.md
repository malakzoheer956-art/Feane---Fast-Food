# Feane - Fast Food Restaurant Website 🍔

---

## 📄 Project Description
A comprehensive web project for a fast-food restaurant named **"Feane"**. This project demonstrates responsive design, interactive menus, and structured booking tables. It is fully hosted on GitHub Pages for preview.

## 🛠 Technologies Used
* **HTML5**: For semantic layout and structured pages.
* **CSS3**: For custom animations, modern UI styling, and responsive layout.
* **JavaScript**: For interactive functionalities.
* **Git & GitHub**: For version control and project hosting.

## 🔗 Live Demo
Experience the live website directly through your browser here:  
👉 [View Live Project](https://malakzoheer956-art.github.io/Feane---Fast-Food/html%20pages/index.html)

## ⚙️ Key Features
* **Interactive Food Menu**: Clearly categorized pages for Burgers, Pizza, and Pasta.
* **Reservation Form**: Sleek booking form interface.
* **Fully Responsive**: Optimized for mobile, tablet, and desktop screens.
