# Feane - Fast Food Restaurant Website 🍔



---

## 📄 Project Description
A comprehensive web project for a fast-food restaurant named **"Feane"**. This project was developed as part of a technical training to apply modern front-end development skills and professional project management using Git and GitHub. It demonstrates advanced code organization, branch management, and live deployment.

## 🛠 Technologies Used
* **HTML5**: For semantic page structure (Menu, Booking, Reviews).
* **CSS3**: For modern, responsive UI design and animations.
* **Git**: For version control (Implemented 5+ Commits).
* **GitHub**: For repository hosting and collaboration (Issues & Pull Requests).
* **GitHub Pages**: For live cloud hosting and deployment.

## 🔗 Live Demo
Experience the live website here:
[View Live Project](https://malkordi4-dev.github.io/github-training-project/html%20pages/index.html)

## ⚙️ Key Features
* **Interactive Menu**: Dynamic display of Burgers, Pizza, and Pasta categories.
* **Reservation System**: A functional UI for table booking services.
* **Responsive Design**: Fully optimized for Mobile, Tablets, and Desktops.

## 🚀 How to Run Locally
To run this project on your local machine, follow these steps:

1.  **Clone the repository:**
    ```bash
git clone [https://github.com/malkordi4-dev/github-training-project.git](https://github.com/malkordi4-dev/github-training-project.git)    ```
2.  **Navigate to the project folder:**
    ```bash
    cd github-training-project
    ```
3.  **Open the website:**
    Locate the `html pages` folder and open `index.html` in any modern web browser.
